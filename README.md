# SD Day1 MovieApp

이 저장소는 영화 정보를 관리하는 간단한 CRUD 애플리케이션입니다.  
Spring Boot와 H2 Database를 활용하여 검색 기능을 포함한 기능 구현을 목표로 합니다.

---
